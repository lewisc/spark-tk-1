LogisticRegressionModelWithFrequency

class LogisticRegressionModelWithFrequency extends GeneralizedLinearModelWithFrequency with ClassificationModel with Serializable with Saveable

Classification model trained using Multinomial/Binary Logistic Regression.

Extension of MlLib's logistic regression model that supports a frequency column. The frequency column contains the frequency of occurrence of each observation.
