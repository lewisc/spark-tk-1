HistogramAggregator

case class HistogramAggregator(cutoffs: List[Double], includeLowest: Option[Boolean] = scala.None, strictBinning: Option[Boolean] = scala.None) extends GroupByAggregator with Product with Serializable

Aggregator for computing histograms using a list of cutoffs.

The histogram is a vector containing the percentage of observations found in each bin
