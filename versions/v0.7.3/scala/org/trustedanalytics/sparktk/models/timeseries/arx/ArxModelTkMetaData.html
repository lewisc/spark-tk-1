org.trustedanalytics.sparktk.models.timeseries.arx

ArxModelTkMetaData

case class ArxModelTkMetaData(timeseriesColumn: String, xColumns: Seq[String], yMaxLag: Int, xMaxLag: Int, noIntercept: Boolean, c: Double, coefficients: Array[Double]) extends Serializable with Product

TK Metadata that will be stored as part of the ARX model

timeseriesColumn

Name of the column that contains the time series values.

xColumns

Names of the column(s) that contain the values of previous exogenous regressors.

yMaxLag

The maximum lag order for the dependent (time series) variable.

xMaxLag

The maximum lag order for exogenous variables.

noIntercept

A boolean flag indicating if the intercept should be dropped.

c

an intercept term, zero if none desired, from the trained model

coefficients

coefficient values from the trained model
